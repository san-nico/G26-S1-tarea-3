# Documentación - Tarea 3

Este repositorio contiene el desarrollo de una landing page enfocada en la implementación de CSS Grid y mejoras de responsividad para distintos dispositivos.

El proyecto busca optimizar la estructura visual y distribución de los elementos mediante técnicas modernas de maquetación en CSS, mejorando la adaptabilidad del sitio y la experiencia de usuario.

---

## Contenidos implementados

- Uso de CSS Grid para estructuración del layout.
- Diseño responsive para diferentes resoluciones.
- Organización visual adaptable en desktop, tablet y mobile.
- Mejora en la distribución y alineación de componentes.

---

## Objetivo de la tarea

Aplicar CSS Grid como sistema principal de maquetación para construir una interfaz más flexible, ordenada y responsive, manteniendo una correcta separación de responsabilidades entre HTML y CSS.
